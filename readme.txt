== Description ==

Notification Runner is a WordPress plugin to disable successful update notification emails for core updates, plugins, and themes. It is light-weight and performant.

== Frequently Asked Questions ==

= Does Notification Runner disable UNSUCCESSFUL update emails? =

No, Notification Runner only disables SUCCESSFUL update notification emails for core updates, plugins, and themes.

You will still receive unsuccessful update notification emails for core updates, plugins, and themes.
